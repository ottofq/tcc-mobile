 ## 📋 Funcionalidades
 - Consultar e avaliar o cardápio do dia
 - Consultar os avisos lançados pelo RU
 - Preencher o questionário elaborado pelo setor de nutrição
 - Verificar informações gerais sobre o funcionamento do RU
